CreoSketch

What we did

CreoSketch is a tool for prototyping UI which is easy to use so you could reveal creativity without any obstacles. CreoSketch makes it easier to build simple, high precision prototypes that look like the real thing. CreoSketch helps you quickly and precisely simulate how an application will look and function without writing any code. Software has a simple and functional interface, which makes it similar to RAD UI tools, such as Visual Studio, Delphi, etc.

Key features:

  • UI prototyping.
  • CreoSketch allows to make mind mapping.
  • UML which offers a standard way to visualize a system's architectural blueprints, including different elements.
